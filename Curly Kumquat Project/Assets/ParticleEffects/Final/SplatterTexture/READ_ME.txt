READ ME

This is a SBSAR texture file.  Don't be alarmed, they're quite easy to use.  An SBSAR file is also know as a Substance file.  A Substace file can be tweaked at run time and is a much smaller file size.  It contains multiple textures within it.  You can create multiple materials by clicking on the plus sign in the top right of the inspector.  

The power of this texture comes from the ability to make the exact texture you want by tweaking parameters in the inspector.

The maps that come with this texture are as follows: Albedo, Metallic with Roughness in the alpha, Normal, Specular, And Emission.
